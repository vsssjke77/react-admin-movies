# react-admin-movies
Приложение для просмотра, создания и редактирования данных о фильмах.
## Сборка и запуск:
1. Скачать проект (два способа):
   - Склонировать проект ` git clone https://github.com/vsssjke77/react-admin-movies.git` в терминале нужной директории
   - Скачать проект как архив и разархивировать в нужную директорию
2. Зайти в папку с проектом (в папку, в которой находится package.json) и открыть в нем терминал
3. Установить зависимости:<br>
   `npm install` <br> `npm install -g json-server` <br>
4. Запустить сервер:<br>
   `json-server --watch db.json`
5. Запустить npm:
   - В терминале той же директории прописать `npm start -port 3001`
   - Далее получаем сообщение, о том что на порте 3000 происходит процесс, и предложение сменить порт => Нажимаем 'Y'<br>

6. *Наслаждаться работой программы*<br>
**Важно: <br> json-server и npm start должны быть на разных портах (:3000 и :3001 соответственно), иначе проект не будет работать**
